What it does

Your gameplay, tracked automatically

Set it up once. From that point on, everything you do in Star Citizen appears in your SC Bridge profile without any manual steps.

🔍

Auto Log Detection

No path browsing. Companion finds your Game.log via launcher, process, registry, or drive scan — whichever works first.

📋

57 Event Patterns

Logins, ships, contracts, money transfers, crimes, quantum travel, injuries, party events — and more, all parsed automatically.

🔄

SC Bridge Sync

OAuth 2.0 authentication. Events sync to your SC Bridge profile every 10 seconds, deduplicated and batched efficiently.

📊

Live Dashboard

Dark-themed React UI with a real-time event feed showing everything happening in your session as it happens.

👥

Friends List

See which of your Spectrum friends are online and active in the game right now.

📥

System Tray

Runs quietly in the background. Minimize to tray, restore on click. Out of the way when you don't need it.

🔔

Self-Updating

Checks for new releases every 4 hours. Update with one click — no re-downloading, no reinstalling.

🤖

WingmanAI Integration

Writes a JSONL event log alongside the app — ready for WingmanAI skill consumption.

The process

Running in five steps

Download, connect once, then forget about it. Companion does the rest.

1

Download

Grab the portable EXE or MSI installer from GitHub Releases. Run it from anywhere — no installation required for the portable version.

2

Connect to SC Bridge

Click Settings → Connect and authorize your SC Bridge account via OAuth in your browser. One-time setup.

3

Launch Star Citizen

Start the game. Companion automatically finds your Game.log and begins tailing it immediately.

4

Play Normally

Every 10 seconds, your in-game activity is parsed, deduplicated, and synced to your SC Bridge profile.

5

Check the Dashboard

Open the Companion window any time to see your live event feed and which friends are currently online.

Good to know

Notes & expectations

SmartScreen warning SC Companion is not yet code-signed. Windows may show a SmartScreen warning on first launch. Right-click the file → Properties → check Unblock → OK. This goes away once signing is in place.
Read-only log access Companion only reads Game.log — it never writes to or modifies any Star Citizen game files, settings, or memory.
Requires SC Bridge account Events sync to scbridge.app. You'll need a free SC Bridge account to connect. Local JSONL logging works without an account.
Developers

Run from source

git clone https://github.com/SC-Bridge/sc-companion
cd sc-companion
go run main.go

Requires Go 1.22+ and Windows 10 / 11. For live-reload dev mode, run wails dev with Wails v2 installed. Build a distributable with wails build.
